## 快速开始

```bash

# 访问：https://app.inference.sh/settings/keys
infsh login --key YOUR_API_KEY

# 2. 运行生成脚本
cd /Users/ajtoy/retro-pop-ppt/demos/yc-intro
./generate-all.sh

# 3. 查看结果
open images/
```

## 手动生成单个图片

```bash
# 例如：生成复古波普风格英文版
infsh app run falai/flux-dev-lora --input '{
  "prompt": "Retro pop art style PPT slide, 1970s magazine aesthetic, flat design with thick 3px black outlines, cream beige background, Title: What is Y Combinator, Subtitle: The Worlds Most Famous Startup Accelerator, Stats: 2005 4000+ $600B+, salmon pink sky blue mustard yellow accents, geometric decorations, bold typography, --ar 16:9 --style raw",
  "aspect_ratio": "16:9"
}'

# 保存到文件
infsh app run falai/flux-dev-lora --input '...' --save result.json
```

## 故障排除

### 问题：需要登录
```bash
infsh login --key YOUR_API_KEY
```

### 问题：模型不可用
换用其他模型：
```bash
# 使用 Seedream 4.5 (高质量)
infsh app run bytedance/seedream-4-5 --input '...'

# 使用 Grok Imagine
infsh app run xai/grok-imagine-image --input '...'
```

### 问题：文字渲染不准确
添加 `--v 6` 或使用专门模型：
```bash
# 使用 Reve (擅长文字渲染)
infsh app run falai/reve --input '...'
```
