# PPT Slide Prompts - 11 Styles

直接复制 Prompt 到 nanobanana2 (gemini-3.1-flash-image-preview) 使用。

**推荐设置:**
- Model: `gemini-3.1-flash-image-preview` (nanobanana2)
- Resolution: `2048*1152` (2K 16:9)
- Aspect Ratio: `16:9`

## 使用方法

1. 复制任意风格 Prompt

## 自定义内容建议

### 标题
- 保持简洁，8 个单词以内
- 例："What is Y Combinator"

### 副标题
- 一行解释，12 个单词以内
- 例："Startup Accelerator"

### 关键数据
- 3-5 个数据点
- 例："2005, 4000+, $600B"

### 金句
- 简短有力的总结
- 例："Startups are hard."

## 技巧

1. **直接修改文字** - Prompt 中已预留文字位置描述
2. **保持 Prompt 简洁** - AI 会自动处理文字渲染
3. **指定 16:9** - 确保 PPT 格式
4. **多试几次** - 同样 Prompt 可能产生不同结果
